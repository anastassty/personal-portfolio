# Personal Portfolio Website

I built this portfolio to showcase my projects, skills, and experience as a Computer Science student.

## 🔗 Live Website

https://anastasty-personal-portfolio.vercel.app/

## ⚙️ Tech Stack

* React
* Tailwind CSS
* Vite
* Vercel

## ✨ Features

* Responsive design
* Project showcase
* About & experience sections
* Clean UI

## 🚀 Run Locally

```bash
git clone https://github.com/anastassty/personal-portfolio.git
cd personal-portfolio
npm install
npm run dev
```
